Student Management System

Kullanılan Konular

Class ve Object

Encapsulation

Inheritance

Abstract Class

Interface

ArrayList

Proje Dosyaları

Person.java → Temel kişi sınıfı

Student.java → Öğrenci sınıfı

NotHesaplama.java → Geçme / kalma hesaplama

Main.java → Programın çalıştığı ana sınıf
